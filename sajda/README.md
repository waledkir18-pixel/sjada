# 🏠 موقع العائلة — Family Site

## هيكل المجلدات

```
family_site/
├── app.py                    ← الملف الرئيسي
├── requirements.txt          ← المكتبات المطلوبة
├── data/
│   ├── users.json            ← يُنشأ تلقائياً عند أول تشغيل
│   └── products.json         ← يُنشأ تلقائياً عند إضافة منتج
└── templates/
    ├── base.html             ← القالب الأساسي
    ├── auth/
    │   ├── login.html        ← صفحة الدخول
    │   └── signup.html       ← صفحة التسجيل
    ├── dashboard/
    │   └── home.html         ← بوابة الاختيار
    ├── admin/
    │   └── panel.html        ← لوحة الأدمن
    ├── store/
    │   └── index.html        ← متجر شراشف الصلاة
    └── web_design/
        └── index.html        ← صفحة تصميم المواقع
```

## التثبيت والتشغيل

```bash
# 1. تثبيت المكتبات
pip install -r requirements.txt

# 2. تشغيل الموقع
python app.py
```
